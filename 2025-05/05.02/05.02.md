# 📝 알고리즘 문제 풀이 - 2025-05-02

## 문제 📖

- **문제 링크:** [백준 - AC](https://www.acmicpc.net/problem/5430)

- **문제 유형:** 문자열 (String), 구현 (Simulation)

- **난이도:** Gold.5

- **풀이 시간:** ⏳ 1시간 - 실제 걸렸던 시간

## ✍ 접근 방식

- 배열을 파싱하여 빈 배열은 건너뛰고, 대괄호 분리 후 쉼표 구분.
- 각 명령을 처리하는 함수 생성.
  - R: 뒤집기 명령어 처리.
  - D: reverse 상태 체크 후 첫 요소 제거.
    - 빈 배열 시 예외 처리.
  - 결과 생성 후 반환.
- 반환된 결과 출력.

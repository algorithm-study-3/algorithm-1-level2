# 📝 알고리즘 문제 풀이 - 2025-05-15

## 문제 📖

- **문제 링크:** [백준 - 감소하는수](https://www.acmicpc.net/problem/1038)

- **문제 유형:** 백트래킹

- **난이도:** 골5

- **풀이 시간:** ⏳ 45분

## ✍ 접근 방식

- 제일 큰 감소하는 수는 9876543210 10자리다.
- 1~10 자리를 순회하며 자리 수에 해당하는 감소하는 수를 찾아본다.
- 감소하는 수인지 확인하고 오름차순 정렬한다.
- N에 해당하는 수를 찾고 없으면 -1을 console.log 한다.

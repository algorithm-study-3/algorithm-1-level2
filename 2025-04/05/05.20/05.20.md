## ✍ 접근 방식

- 각 배열의 행 순회
- 행에서의 1개의 열 값을 2진법으로 변환
- | 연산자
  - 0과 1 만나면 1
  - 1과 0 만나면 1
  - 1과 1 만나면 1
  - 0과 0 만나면 0
- 1은 # 0은 ' ' 변환

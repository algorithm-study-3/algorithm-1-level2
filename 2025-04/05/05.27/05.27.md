# 📝 알고리즘 문제 풀이 - 2025-05-27

## 문제 📖

- **문제 링크:** [프로그래머스 - 뉴스 클러스터링](https://school.programmers.co.kr/learn/courses/30/lessons/17677)

- **문제 유형:** 구현

- **난이도:** Level2

- **풀이 시간:** ⏳ 26분

## ✍ 접근 방식

- 각각의 문자열 lowerCase로 변경
- 2글자씩 잘라서 알파벳만 포함하는지 확인하고 배열로 만들기
- 교집합 구하고 각각의 배열에서 교집합에 해당하는 값들 빼기
- 합집합의 크기 구하기
- 합집합이 0이라면 65536 반환
- 아니면 교집합 / 합집합
